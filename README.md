# E-Commerce Customer Spending Analysis 📊

This project focuses on analyzing an e-commerce customer dataset to understand user behavior and spending patterns using **Python data analysis and visualization libraries**.

## 📌 Project Overview
The notebook performs **exploratory data analysis (EDA)** on an e-commerce dataset to identify relationships between customer activity (such as time spent on website) and yearly spending. Visualizations are used to gain insights that can help businesses improve customer engagement and revenue.

## 🛠️ Technologies Used
- Python  
- Pandas  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

## 📂 Dataset
The dataset contains customer-related features such as:
- Time on Website  
- Time on App  
- Length of Membership  
- Yearly Amount Spent  

> **Note:** Dataset file used in the notebook: `ecommers.csv`

## 📊 Key Analysis Performed
- Data loading and inspection  
- Descriptive statistics  
- Visualization using:
  - Joint plots
  - Regression plots
- Relationship analysis between customer activity and yearly spending  

## 📈 Visualizations
Seaborn is used to create insightful plots such as:
- Regression-based joint plots
- Feature relationship analysis  

These plots help identify trends and correlations in customer behavior.
